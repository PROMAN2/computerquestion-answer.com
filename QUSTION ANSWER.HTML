SHORT QUSTION ANSWER

Q1. WHAT IS HTML?

ANS. HTML STANDS FOR HYPERTEXT MARKUP LANGUAGE IT IS A MARKUP LANGUAGE THAT DESCRIBES THE STRUCTURE OF TE WEB PAGE . IT ALLOWS US TO CREATE WEB PAGE THAT CONTAIN PARAGRAPHS,HEADING,LINK AND BLOCK QUETES.

Q2. DEFINE CONTAINER TAGS?

ANS. THE TAGS THAT HAVE BOTH OPENINGS AND CLOSEING TAGS ARE CALLED CONTANIER TAGS.

Q3. EXPLAIN THE PURPOSE OF HTML TAGS ?

ANS. THE HTML TAG TELLS THE TEXT CONTAINED BETTWEEN HTML AND /HTML IS A WEB PAGE AND CAN BE VIEWED USING A WEB BROWSER .

Q4. WHAT DO YOU MEAN BY HEAD TAG?

ANS. THE HEAD TAG DEFINES HEADER AREA OF YOUR WEB PAGE . THE INFORMATION GIVEN IN HEAD TAG TELLS THE COMPUTER THAT THIS INFORMATION IS NOT TO BE SHOWN ON THE WEB PAGE.
