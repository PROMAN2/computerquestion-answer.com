Q1. WHAT ARE THE RULES REQIRED FOR HTML TAGS?

ANS. THE RULES REQUIRED FOR HTML TAGS:-

1. CONTAINER TAGS SHOULD ALWAYS BE CLOSED PROPERLY.

2. VALUES GIVENS TO THE ATTRIBULE SHOULD BE ENCLOSED WITHIN DOUBLE QUOTES ("")

3. TAG NAME SHOULD NOT CONTAIN SPACE.

4. THERE SHOULD BE NO SPACE BETWEEN IN A TAG.

5. TAG MUST BE NESED CORRECTLY.
